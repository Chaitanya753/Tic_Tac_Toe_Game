## 🚀 Tic Tac Toe Game

A fully interactive two-player Tic Tac Toe game built using HTML, CSS, and JavaScript.
This project demonstrates DOM manipulation, event handling, responsive design, sound integration, and game logic implementation.

The game allows two players to enter their names, track scores, enjoy click and win sound effects, and view glowing win highlights for better user experience.

## 📦 Installation & Setup

No special installation required.

Option 1: Run Locally
# Clone the repository
https://github.com/Chaitanya753/Tic_Tac_Toe_Game.git


# Open the project folder
cd tic-tac-toe-game

# Open index.html in browser
Option 2: Use Live Server (Recommended)

Open project in VS Code

Install Live Server extension

Right click on index.html

Click Open with Live Server
